


# Expense Titan

![Version](https://img.shields.io/badge/version-v1.0.0-blue) ![License](https://img.shields.io/badge/license-MIT-green) ![Size](https://img.shields.io/badge/size-~35KB-lightgrey)

### A zero-knowledge, offline-first personal finance tracker in a single HTML file.

Expense Titan is designed for users who want granular control over their financial data without the bloat of cloud services, subscription fees, or privacy-invasive tracking.

---

## Why Expense Titan?

Modern finance tools are bloated. They require logins, sync to servers you don't control, and lock essential features behind paywalls. 

Expense Titan exists because tracking your spending should be a utility, not a product. It stores your data locally in your browser. No one else sees it. No one else owns it.

### The Utility Gap

| **Before** | **After** |
| :--- | :--- |
| Spreadsheets are ugly and hard to manage on mobile. | A responsive interface that looks good on any screen. |
| Cloud apps sell your data to advertisers. | 100% LocalStorage. Zero network traffic for data. |
| "Premium" features are locked behind $10/month. | Full access to Analytics, Budgeting, and Exporting for free. |
| Requires complex installation or Python environments. | A single `index.html` file. Double-click to run. |

## Quick Start (Under 60 Seconds)

You do not need to install Node.js, Python, or a database.

### Option A: Use Online
1.  Click the **Web App Link** below.
2.  Start tracking transactions.
3.  Bookmark the page.

### Option B: Use Offline (Recommended)
1.  Download the **Direct Download Link** below.
2.  Save `index.html` to your computer or mobile device.
3.  Open the file in any modern web browser.
4.  **You are done.** The app is now self-contained and works without an internet connection.

## Features

*   **Privacy First:** All data is persisted in your browser's LocalStorage. No external APIs, no tracking pixels.
*   **Visual Analytics:** Generate trend lines and category breakdowns using integrated Chart.js.
*   **Budget Control:** Set monthly limits per category with visual progress bars.
*   **Theming Engine:** 12 built-in themes (Light, Dark, Neon, Dracula, etc.) to match your OS preference.
*   **Data Portability:** Export your full ledger to **CSV** (Excel/Sheets) or **JSON** (Backup/Restore) instantly.
*   **Smart Defaults:** Remembers your last 5 transaction descriptions for rapid entry.

## Data Safety

Your data never leaves your device unless you choose to export it.
*   **Storage:** Browser `localStorage`.
*   **Backup:** Go to **Settings > Data Management > Full Backup** to download a `.json` file.
*   **Restore:** Upload your `.json` backup to restore your state on a new device or browser.
